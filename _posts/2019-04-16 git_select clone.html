ex) 해당 repo

clone_flie만을 clone 하고 싶을 경우 (sparse checkout) 기능참고

  1. 1. sparse Checkout 이 가능하도록 설정
    Git code :
    git config core.sparseCheckout true
  2. 2. remote 추가
    Git code :
    git remote add -f origin <REMOTE_URL>
  3. 3. 원하는 폴더 위치 추가 (echo 이후에 "" 사이입력)
    Git code :
    echo "test1/clone_flie" >> .git/info/sparse-checkout
  4. 4. 파일 풀 받기
    Git code :
    git pull origin master

해당내용은

클론내용 원본블로그

위사이트를 참고 하였음 (삭제될까봐 미리 메모겸 게시글 등록)
